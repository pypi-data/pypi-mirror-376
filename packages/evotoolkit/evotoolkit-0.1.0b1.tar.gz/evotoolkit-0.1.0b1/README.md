# EvoTool

A Python library for evolutionary optimization applications with GPU acceleration support.

## Installation

```bash
pip install evotool
```

## Development

```bash
# Clone the repository
git clone https://github.com/pgg3/evotool.git
cd evotool

# Install in development mode
pip install -e .
```

## Usage

```python
import evotool

# Your code here
```
